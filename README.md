# Redes-chat-socket

### DependÃªncias: 
* docker
* npm/node
* yarn
### Rodando:
* git clone git@github.com:luucas-melo/Redes-chat-socket.git
* cd Redes-chat-socket/
* Abra dois terminais:
* No primeiro:
```
cd server/
docker-compose up ou docker compose up
```
* No segundo: 
```
cd client/
yarn
yarn dev
```
